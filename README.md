# AI-Based Breast Cancer Detection

## Overview
This project focuses on early breast cancer detection using thermal images and hybrid AI techniques. The system combines handcrafted image features and deep learning models for accurate classification.

## Technologies Used
- Python
- OpenCV
- TensorFlow / Keras
- Scikit-learn
- ResNet50
- MobileNetV2

## Features
- Image preprocessing using CLAHE
- Handcrafted feature extraction
- Deep learning feature extraction
- PCA dimensionality reduction
- SMOTE balancing
- SVM classification
- Grad-CAM visualization

## Model Accuracy
- Hybrid Model Accuracy: 96.7%
- ROC-AUC Score: 0.988

## Dataset
Breast thermal image dataset used for classification.
